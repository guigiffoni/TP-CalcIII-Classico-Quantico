# Objetivo do Repositório

- Armazenar as fontes consultadas do trabalho;
- Código fonte/LaTeX utilizado para as animações;
- Ferramentas utilizadas;
- Roteiro do vídeo;
- Documentação relacionada;
- E outros quaisquer itens.

A distribuição das tarefas dentro do grupo ficará automaticamente 
registrada no repositório mediante a sua conta atrelada no commit.

as pastas roteiro-original e roteiro-separado, servem para arquivamento de versões anteriores.

a pasta fontes é onde ficam os pdfs contendo as fontes, entretanto por questões de licenciamento, não podemos postar diretamente os pdfs (exceto quando o mesmo for open access), mas ao menos precisamos ter uma relação desses documentos, é aí o que _relação.txt_ tem como objetivo.
